### Chevron example:
Downward Chevron:
```js
<Chevron />
```

Left Chevron:
```js
<Chevron direction='left' />
```

Right Chevron:
```js
<Chevron direction='right' />
```

Animated Left Chevron:
```js
<Chevron animated direction='left' />
```

Animated Right Chevron:
```js
<Chevron animated direction='right' />
```
